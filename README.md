# stackpilot

開発者向けワークスペース型デスクトップブラウザ（Electron + React + Vite）。

## セットアップ

```bash
pnpm install
pnpm dev
```

## スクリプト

- `pnpm dev`: renderer + electron 起動
- `pnpm build`: renderer/electron ビルド
- `pnpm test`: unit test (Vitest)
## ドキュメント

- `docs.md`: MVPアーキテクチャ設計メモ
- `docs_uiux_spec_ja.md`: UI/UX仕様（開発者向けワークスペースブラウザ）
